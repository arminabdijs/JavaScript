## âœ… Section 1: Introductory (Completed)

This section covers the essential and core concepts of JavaScript.
It is designed for beginners and anyone who wants to build a strong foundation in JavaScript.

### Main Topics:

- Variables and Data Types
- Type Conversion
- Operators  
  (Mathematical, Logical, Comparative, Assignment)
- Conditions and Control Flow  
  (`if`, `switch`)
- Loops  
  (`for`, `while`, `do...while`, `for in`, `for of`)
- Functions  
  (Declaration, Expression, Arrow, Anonymous, IIFE)
- Scope and Hoisting
- Truthy and Falsy values
- Arrays and Array Methods  
  (`map`, `filter`, `reduce`, `find`, `some`, `every`, `forEach`)
- Objects and Object Destructuring
- Spread Syntax and Rest Operator
- Primitive vs Reference types
- Pure Functions and Higher-Order Functions
- Callback Functions
- Async vs Sync
- Promises and async/await
- Working with APIs
- JSON
- Modules and export/import
- Object-Oriented Programming (OOP)
- Error Handling (`try...catch`)
- setTimeout and setInterval
